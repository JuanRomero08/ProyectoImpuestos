# ProyectoImpuestos
Este proyecto es una aplicación en Java que permite calcular impuestos de vehículos de forma rapida, esta desarrollado siguiendo la arquitectura MVC (Modelo-Vista-Controlador) para mantener una buena organizacion y diseño del proyecto.

El sistema permite al usuario ingresar información del vehículo, como marca, modelo, año, cilindraje, avalúo comercial y tipo de uso, y calcula automáticamente el impuesto correspondiente considerando reglas como tarifas por rango de avalúo, ajustes para vehículos de uso público y descuentos por antigüedad.

El proyecto incluye las siguientes características principales:

Interfaz gráfica intuitiva y fácil de usar.

Validación y manejo de errores en la entrada de datos.

Cálculo de impuestos centralizado en la clase CalculadoraImpuestos.

Registro y cálculo de la antigüedad del vehículo dentro del modelo Vehiculo.
